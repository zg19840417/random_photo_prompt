# 规则2：关键词扩写模式选项扩充规范

## 文档职责

本文只负责“规则2：关键词扩写模式”的选项扩充格式和写作约束，供其他 AI 扩充规则2选项时读取。

本文不负责规则1六维度随机池、不负责图生视频动作选项、不负责远程访问配置。相关文档：

- `docs/AI_CONTEXT.md`：项目总索引和当前开发上下文。
- `docs/PROMPT_GENERATION_RULES.md`：规则1文生图提示词生成规则。
- `docs/VIDEO_PROMPT_OPTION_GUIDE.md`：图生视频动作提示词选项扩充规则。

## 一句话交给其他 AI

请先阅读 `docs/KEYWORD_EXPANSION_OPTION_GUIDE.md`，然后按本文格式扩充“规则2：关键词扩写模式”的选项；每条只写可直接拼进提示词的自然中文片段，不写解释，不写 JSON，不写编号。

## 模式定位

规则2是独立于规则1六维度随机池的“关键词扩写模式”。它不让用户手动输入关键词，而是由程序自动抽取主题、镜头、姿势、表情、服装、场景、光线和扩写规则片段，再组装成完整文生图正向提示词。

规则2仍然必须读取界面上的：

- 档位：一档、二档、三档、四档
- 镜头：头部、上半身、半身、大半身、全身

规则2改变的是提示词写法和扩写风格，不改变这些 UI 选项的含义。

## 项目级长度规则

最终正向提示词控制在 **800 个中文字符以内**。扩写选项不要写成长段散文。每条选项应短、具体、可直接入画。

## 具体可画规则

规则2禁止依赖抽象形容词让模型自行理解。所有选项必须写成模型能直接画出来的动作、颜色、眼睛状态、材质或空间物体。

不要写：

- 姿势主动
- 视线有挑逗感
- 眼神像在勾引前方视线
- 亮色耳饰、亮色衣料、亮色指甲
- 浅色地面、浅色露台、浅色短裤
- 表情明确、主体清晰、构图明确、材质真实

要写成：

- 身体向前景靠近，半眯狐眼直看前方
- 半眯狐眼直看前方，眼尾上挑
- 湖蓝色耳饰、玫粉或湖蓝衣料、湖蓝色指甲
- 米白地砖、淡金色沙面、米白短裤
- 发丝、指尖、细银链和地面反光清楚可见

程序层也会在规则2最终输出前做一次抽象词具体化处理。后续扩充选项仍然应该从源头避免抽象词，而不是依赖后处理兜底。

## 最终输出格式

规则2最终生成的正向提示词也必须按段落提行展示。每个维度单独一段，段落之间空一行。

推荐顺序：

1. 镜头与档位风格
2. 角色固定描述
3. 穿着或尺度相关外观，一档和二档保留，三档和四档跳过穿着
4. 姿势与表情
5. 场景与光线
6. 镜头质感收尾包

不要把规则2最终提示词压成一整段逗号长句。其他 AI 扩充选项时仍然只写自然中文片段，不需要自己加维度标题。

## 档位规则

### 一档

一档重点是多样性，而不是性感强度。

应扩充：

- 时尚杂志
- 阳光生活方式
- 商业人像
- 城市街拍
- 度假写真
- 彩色棚拍
- 运动休闲
- 清新学院
- 轻奢套装
- 多样场景和完整穿着

不要让一档显得像私房写真或成人诱惑。可以有好看的身材和自然魅力，但重点是造型、情绪、色彩、场景和真实拍摄瞬间。

### 二档

二档重点是性感诱惑，但仍然有明确穿着结构。

应扩充：

- 性感 glamour 写真
- 明亮私房感
- 轻薄鲜艳穿着
- 半透明薄纱、吊带、短款上衣、泳装感套装、局部蕾丝
- 锁骨、胸腰曲线、臀腿线条、脚部或手部强透视
- 勾引、俏皮挑衅、坏笑、得意嘲笑、半眯眼直视镜头
- 少量有效配饰：锁骨链、脚链、细链、透明手套、亮色指甲

二档不要写显式性行为，也不要写完全裸体。穿着可以少、薄、透、贴身，但要能看出衣物结构。

### 三档

三档沿用二档的明亮 glamour、性感诱惑和私房张力逻辑，但不扩充穿着关键词。

应扩充：

- 二档同级别的性感姿势、直接眼神和镜头压迫感
- 肌肤光泽、身体张力、胸腰臀腿曲线
- 勾引、俏皮挑衅、坏笑、得意嘲笑、半眯眼直视镜头

三档不需要扩充穿着关键词，也不要转成原 NSFW 语气。

### 四档

四档重点是成人私房、身体表现和更强情欲氛围。

应扩充：

- 成人私房写真
- 更大胆的身体姿势
- 直接眼神和镜头压迫感
- 肌肤光泽、身体张力、胸腰臀腿曲线
- 大幅扭腰、开腿、跪趴、仰躺、侧躺、俯身靠近镜头、强透视展示身体
- 挑衅坏笑、轻蔑笑、湿润直视、半睁狐眼、主动勾引镜头

四档不需要扩充穿着关键词。四档重点转向身体姿势、肌肤光泽、成人私房氛围和镜头距离。

## 镜头规则

扩充选项必须尊重镜头范围：

- 头部：肩膀及以上镜头，头顶完整。只能写脸、头发、颈部、肩膀、嘴唇、眼神、手指靠近脸部等。
- 上半身：胸部及以上镜头，头顶完整。可以写肩颈、锁骨、完整胸部、上腰边缘、手部动作。
- 半身：腰部及以上镜头。可以写胸、腰、手臂、上半身扭转、腰线。
- 大半身：大腿以上镜头。可以写臀腿线、膝盖、大腿、小腿上段，但不要强调脚掌细节。
- 全身：完整身形入镜。可以写站、坐、跪、躺、跳、趴、侧躺、脚下具体地面、完整身体动作。

全身镜头建议写具体地面，例如“脚踩在浅金沙面上”“脚踩在湖蓝泳池瓷砖上”“脚踩在彩色街道路面上”。不要写“脚下地面清楚”这种空话。

## 可扩充分类

### scale_style

按档位写整体风格。

示例：

```text
清爽时尚杂志写真，画面重点是造型、色彩和自然情绪
性感glamour写真，画面以诱惑眼神、锁骨、胸腰曲线和腿部线条为重点
成人私房写真，情欲氛围强，画面重点是身体曲线、直接眼神和大胆姿势
```

### scale_pose

按档位写姿势和表情倾向。

示例：

```text
姿势以自然转身、走动、坐姿、靠坐、回头、伸手互动为主
姿势可以前倾、侧扭、跪坐、趴靠、侧躺、回头看镜头或手指靠近嘴唇
姿势可以大幅度扭腰、开腿、跪趴、仰躺、侧躺、俯身靠近镜头或强透视展示身体
```

### scale_outfit

只给一档和二档写穿着。三档和四档不写穿着。

示例：

```text
穿着完整但多样，可以是连衣裙、短上衣、牛仔、运动风、度假风或轻奢套装
穿着以半透明薄纱、轻薄吊带、短款亮色上衣、泳装感套装或局部蕾丝为主
```

### theme_keywords

写画面主题和场景方向。

示例：

```text
阳光泳池边写真
夏日海边时尚写真
彩色棚拍甜辣大片
露台阳光私房写真
花园度假风写真
城市街头高饱和写真
酒店阳台明亮写真
```

### pose_keywords

写具体可见动作。不要抽象。

示例：

```text
趴在泳池边回头看镜头，双腿向后轻轻抬起
坐在浅色木质躺椅边，身体转向镜头，右手轻扶长发
脚踩在暖色木质露台地板上，身体向镜头轻轻转过来
靠近镜头伸出右手，手指形成前景强透视
```

如果左右手、左右腿动作不同，必须写明左手、右手、左腿、右腿。不要写“一只手”“另一只手”“一条腿”“另一条腿”。

### expression_keywords

写具体表情，不写“表情明确”这种空话。

示例：

```text
明亮俏皮笑，眼神主动邀请镜头
甜美大笑，像被镜头逗笑的一瞬间
得意挑衅的笑，眼尾上扬直看镜头
带一点怪罪感的笑，像在轻轻责怪镜头
湿润直视镜头，嘴角带轻蔑的挑衅笑
```

### outfit_keywords

只用于一档和二档。写具体颜色、材质、结构。

示例：

```text
柠檬黄短上衣和薄荷绿短裙，颜色明亮清爽
珊瑚粉和湖蓝色泳装感套装，搭配少量金属小配饰
薰衣草紫轻薄上衣和蜜桃橙短裙，材质轻盈
玫粉色亮面吊带和浅色短裤，细小银色配饰点缀
```

避免黑白灰占比过高。不要写不透明 PVC 或厚重皮质衣服。

### scene_keywords

写真实场景内容。道具必须匹配环境。

示例：

```text
蓝色泳池、水面反光、彩色泳池砖和阳光伞组成背景
海边浅金沙面、蓝色海面、彩色遮阳伞和阳光反射铺满背景
城市彩色墙面、街边阳光、浅色地面和远处店招
花园绿色植物、鲜艳花朵、浅色石板地面和自然阳光
```

不要把室内窗帘、白墙、藤椅、玻璃块硬塞进户外海边、泳池、花园、露台。

### light_color_keywords

写光线、颜色、饱和度和皮肤表现。

示例：

```text
温暖阳光，高饱和多色协调，皮肤显白但不过曝
明亮自然光和局部彩色反光，画面对比度高，颜色鲜艳
清爽夏日调色，主色鲜明，辅助色和点缀色彼此协调
高对比商业写真调色，背景鲜艳，人物皮肤保持冷白干净
```

颜色丰富指构图里有多种合理颜色，不是写彩虹，也不是写透明玻璃块。

### subject_detail

写主体细节如何被强化。

示例：

```text
人物与场景关系清晰，动作像照片瞬间被捕捉下来
主体气质鲜明，表情、姿态和造型服务同一个画面主题
人物比例自然，姿势有动态感，画面重点不分散
```

### scene_environment

写环境如何服务画面。

示例：

```text
真实环境铺满画面，背景有可识别的空间层次和生活化细节
场景道具与主题匹配，近景、中景和远景都有自然内容
背景色彩丰富但不抢主体，空间边缘也有真实场景内容
```

### composition_camera

写构图和镜头。

示例：

```text
摄影级构图，主体位置稳定，画面重心明确，透视自然
真实相机视角，人物与环境比例可信，四周没有空白填充感
构图有明确视觉路径，从表情延伸到动作和环境细节
```

### lighting_color

写光影和色彩。

示例：

```text
高对比度高饱和调色，阳光感鲜明，肤色干净透亮
明亮暖光与局部彩色反光配合，皮肤显白但不过曝
色彩搭配积极鲜艳，主色、辅助色和点缀色彼此协调
```

### material_texture

写材质细节。

示例：

```text
布料、皮肤、发丝和地面材质都有真实质感
服装边缘、金属小配饰和发丝高光清楚可见
局部反光来自合理材质，例如水面、瓷砖、金属边或亮面布料
```

### quality_tail

写短质量尾巴，不要盖过具体画面。规则2最终输出更推荐使用短的镜头质感收尾包，而不是把“主体清晰、构图明确、材质真实、画面完成度高”等抽象总结句全部堆进去。

示例：

```text
高饱和杂志大片调色，阳光高光落在皮肤和发丝上，阴影保留清晰层次
glamour写真质感，皮肤带细腻光泽，亮色衣料、细链和发丝高光在阳光下清楚发亮
近距离glamour镜头质感，发丝、指尖、皮肤水光和地面反光清楚，画面私密而鲜艳
```

## 推荐输出格式

让其他 AI 扩充时，按下面格式输出：

```text
【scale_style｜一档】
清爽时尚杂志写真，画面重点是造型、色彩和自然情绪
阳光生活方式大片，场景明亮丰富，人物状态轻松鲜活

【scale_style｜二档】
性感glamour写真，画面以诱惑眼神、锁骨、胸腰曲线和腿部线条为重点
明亮私房写真质感，穿着轻薄鲜艳，性感但仍有明确衣物结构

【scale_style｜三档】
成人私房写真，情欲氛围强，画面重点是身体曲线、直接眼神和大胆姿势
近距离成人私房构图，皮肤光泽、身体张力和镜头压迫感更强

【pose_keywords】
趴在泳池边回头看镜头，双腿向后轻轻抬起
靠近镜头伸出右手，手指形成前景强透视

【scene_keywords】
蓝色泳池、水面反光、彩色泳池砖和阳光伞组成背景
城市彩色墙面、街边阳光、浅色地面和远处店招
```

每个分类可以写很多条。不要编号，不要解释，不要输出 JSON。
